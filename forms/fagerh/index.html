        Avant de commencer

        Merci de compléter le questionnaire dans cet ordre : Identification, Volet RH, Contexte écologique, puis Vos prestations. Les formulaires conditionnels s'affichent automatiquement selon vos choix.

        Mode d'emploi rapide
        1. Cliquez sur Commencer ma saisie.
        2. Complétez les étapes dans l'ordre, avec les boutons Retour et Continuer.
        3. Les sections non pertinentes restent masquées automatiquement selon vos choix.
        4. Cliquez sur Enregistrer régulièrement pour conserver vos réponses.
        5. Pour reprendre une saisie, cliquez sur Reprendre la saisie puis renseignez email + FINESS.
        6. Si un FINESS est déjà utilisé, reprenez la saisie du questionnaire existant avec ce même couple email + FINESS.
        7. Quand tout est finalisé, cliquez sur J'ai terminé ma saisie (vous pourrez revenir en modification si nécessaire).
